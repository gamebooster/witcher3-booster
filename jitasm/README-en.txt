jitasm
x86/x64 JIT Assembler Library
http://code.google.com/p/jitasm/

Introduction
============
jitasm is C++ library for runtime code generation of x86/x64. You can
write the code like a inline assembler.

License
=======
jitasm is open source software. You can distribute it under the terms of
the new BSD license.

Compiler
========
 - VisualC++ 2005 or later
 - GCC 4.1 or later

How to use
==========
You just copy and include jitasm.h. Please see the sample code in
"samples" directory.
